Instructions for the user registration

Die Benutzerregistrierung für Job-Room erfolgt über das System eIAM, der sicheren Identitäts- und Zugriffsverwaltung des Bundes. Dank eIAM können Sie mit einem einzigen Login verschiedene Informatikangebote der Bundesverwaltung nutzen. Da eIAM eine Dienstleistung des Bundesamtes für Informatik und Telekommunikation (BIT) ist, weicht das grafische Design von der Gestaltung auf Job-Room ab. Job-Room bietet Ihnen dank eIAM verschiedene Möglichkeiten, sich als Benutzer zu identifizieren.

Register with email, password and SMS code (CH login)

Verwenden Sie eine E-Mail-Adresse als Benutzername und ein individuelles Passwort, um sich auf Job-Room anzumelden. Um die Sicherheit weiter zu erhöhen, senden wir Ihnen bei jeder Anmeldung einen Code per SMS an Ihr Mobiltelefon. Diese Methode ist für alle Benutzer verfügbar.
Folgen Sie der Anleitung, um sich mit dieser Methode zu registrieren.

Register with SuisseID

If you own a SuisseID you can also use it to log into arbeit.swiss.
Follow the instructions to register with your SuisseID.
Press button to start the user registration.
